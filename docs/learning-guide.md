# C++ 学习指南

## 🎯 学习原则

1. **动手实践优于被动阅读**
   - 每个主题都要写代码
   - 修改示例代码，观察输出
   - 尝试预测程序运行结果

2. **循序渐进**
   - 遵循课程顺序学习
   - 不要跳过基础
   - 每个概念学透了再继续

3. **多维学习**
   - 理解概念
   - 写代码实现
   - 反复复习加深印象

## ⚠️ 常见错误和解决方案

### 1. 编译错误
**问题**：`undefined reference to function`
**原因**：函数声明但未定义
**解决**：检查函数实现，或在 CMakeLists.txt 中添加源文件

### 2. 运行时错误
**问题**：程序崩溃或段错误
**原因**：指针问题或访问无效内存
**解决**：使用调试器，检查指针和边界条件

### 3. 逻辑错误
**问题**：程序运行但输出不对
**原因**：算法或条件有问题
**解决**：添加调试输出，追踪变量值
